Aman CTF
===============

本平台采用ThinkPHP搭建。 本人只用了一个星期的时间，所以该系统功能并不是很丰富。但能够满足CTF平台的正常运行。

##已有功能
 + CTF答题
 + 排行榜
 + 资源库
 + 在线工具
 + 基础防入侵功能（采用的是网上公开的WAF，有能力的后续可以自己改写）
 
##后续待开发功能
 + 积分商城
 + 战队功能
 + 赏金猎人功能
 + 荣誉墙


## 安装

 + 导入ctf.sql
 + 后台/amanmin（可以自己去修改）
 + 账号/密码：admin/admin
